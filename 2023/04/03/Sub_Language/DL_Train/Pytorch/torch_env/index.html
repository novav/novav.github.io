pytorch-lightning

安装pytorch-lightning时一定注意自己的torch是pip安装还是conda安装，pytorch_lightning 安装方式要与torch的安装方式保持一致，否则也会导致你的torch版本被替换。

conda install pytorch-lightning -c conda-forge

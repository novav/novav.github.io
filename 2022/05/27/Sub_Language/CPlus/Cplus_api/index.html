使用以下几种函数判断文件是否存在

  1. #include <fstream> –> 使用ifstream打开文件流，成功则存在，失败则不存在;
  2. #include <stdio.h> –> 以fopen读方式打开文件，成功则存在，否则不存在;
  3. #include <unistd.h> –> 使用access函数获取文件状态，成功则存在，否则不存在
  4. #include <sys/stat.h> –> 使用stat函数获取文件状态，成功则存在，否则不存在

еҶ…йғЁе®һзҺ°жңәзҗҶдёҚеҗҢ
mapпјҡ mapеҶ…йғЁе®һзҺ°дәҶвјҖдёӘзәўвҝҠж ‘пјҲзәўвҝҠж ‘жҳҜвҫ®дёҘж је№іиЎЎвјҶеҸүжҗңзҙўж ‘пјҢвҪҪAVLжҳҜдёҘж је№іиЎЎвјҶеҸүжҗңзҙўж ‘пјүпјҢзәўвҝҠж ‘е…·жңүвҫғеҠЁжҺ’еәҸзҡ„еҠҹиғҪпјҢеӣ жӯӨmapеҶ…йғЁзҡ„жүҖжңүе…ғзҙ йғҪжҳҜжңүеәҸзҡ„пјҢзәўвҝҠж ‘зҡ„жҜҸвјҖдёӘиҠӮзӮ№йғҪд»ЈиЎЁзқҖmapзҡ„вјҖдёӘе…ғзҙ гҖӮеӣ жӯӨпјҢеҜ№дәҺmapиҝӣвҫҸзҡ„жҹҘжүҫпјҢеҲ йҷӨпјҢж·»еҠ зӯүвјҖзі»еҲ—зҡ„ж“ҚдҪңйғҪзӣёеҪ“дәҺжҳҜеҜ№зәўвҝҠж ‘иҝӣвҫҸзҡ„ж“ҚдҪңгҖӮmapдёӯзҡ„е…ғзҙ жҳҜжҢүз…§вјҶеҸүжҗңзҙўж ‘пјҲвјңеҗҚвјҶеҸүжҹҘжүҫж ‘гҖҒвјҶеҸүжҺ’еәҸж ‘пјҢзү№зӮ№е°ұжҳҜе·ҰвјҰж ‘дёҠжүҖжңүиҠӮзӮ№зҡ„й”®еҖјйғҪвј©дәҺж №иҠӮзӮ№зҡ„й”®еҖјпјҢеҸівјҰж ‘жүҖжңүиҠӮзӮ№зҡ„й”®еҖјйғҪвјӨдәҺж №иҠӮзӮ№зҡ„й”®еҖјпјүеӯҳеӮЁзҡ„пјҢдҪҝвҪӨдёӯеәҸйҒҚеҺҶеҸҜе°Ҷй”®еҖјжҢүз…§д»Һвј©еҲ°вјӨйҒҚеҺҶеҮәжқҘгҖӮ

unordered_map: еҶ…йғЁе®һзҺ°дәҶвјҖдёӘе“ҲеёҢиЎЁпјҲд№ҹеҸ«ж•ЈеҲ—иЎЁпјҢйҖҡиҝҮжҠҠе…ій”®з ҒеҖјжҳ е°„еҲ°HashиЎЁдёӯвјҖдёӘдҪҚзҪ®жқҘи®ҝй—®и®°еҪ•пјҢжҹҘжүҫзҡ„ж—¶й—ҙеӨҚжқӮеәҰеҸҜиҫҫеҲ°O(1)пјҢе…¶еңЁжө·йҮҸж•°жҚ®еӨ„зҗҶдёӯжңүзқҖвјҙжіӣеә”вҪӨпјүгҖӮеӣ жӯӨпјҢе…¶е…ғзҙ зҡ„жҺ’еҲ—йЎәеәҸжҳҜвҪҶеәҸзҡ„гҖӮ

зү№зӮ№
map жңүеәҸжҖ§пјҢиҝҷжҳҜmapз»“жһ„жңҖвјӨзҡ„дјҳзӮ№пјҢе…¶е…ғзҙ зҡ„жңүеәҸжҖ§еңЁеҫҲеӨҡеә”вҪӨдёӯйғҪдјҡз®ҖеҢ–еҫҲеӨҡзҡ„ж“ҚдҪң
зәўвҝҠж ‘пјҢеҶ…йғЁе®һзҺ°вјҖдёӘзәўвҝҠд№ҰдҪҝеҫ—mapзҡ„еҫҲеӨҡж“ҚдҪңеңЁlgnзҡ„ж—¶й—ҙеӨҚжқӮеәҰдёӢе°ұеҸҜд»Ҙе®һзҺ°пјҢеӣ жӯӨж•ҲзҺҮвҫ®еёёзҡ„вҫј
зјәзӮ№пјҡ з©әй—ҙеҚ вҪӨзҺҮвҫјпјҢеӣ дёәmapеҶ…йғЁе®һзҺ°дәҶзәўвҝҠж ‘пјҢиҷҪз„¶жҸҗвҫјдәҶиҝҗвҫҸж•ҲзҺҮпјҢдҪҶжҳҜеӣ дёәжҜҸвјҖдёӘиҠӮзӮ№йғҪйңҖиҰҒйўқеӨ–дҝқеӯҳвҪ—иҠӮзӮ№гҖҒеӯ©вјҰиҠӮзӮ№е’Ңзәў/вҝҠ
жҖ§иҙЁпјҢдҪҝеҫ—жҜҸвјҖдёӘиҠӮзӮ№йғҪеҚ вҪӨвјӨйҮҸзҡ„з©әй—ҙ
йҖӮвҪӨеӨ„пјҡеҜ№дәҺйӮЈдәӣжңүйЎәеәҸиҰҒжұӮзҡ„й—®йўҳпјҢвҪӨmapдјҡжӣҙвҫјж•ҲвјҖдәӣ
unorder_map дјҳзӮ№пјҡ еӣ дёәеҶ…йғЁе®һзҺ°дәҶе“ҲеёҢиЎЁпјҢеӣ жӯӨе…¶жҹҘжүҫйҖҹеәҰвҫ®еёёзҡ„еҝ«
зјәзӮ№пјҡ е“ҲеёҢиЎЁзҡ„е»әвҪҙвҪҗиҫғиҖ—иҙ№ж—¶й—ҙ
йҖӮвҪӨеӨ„пјҡеҜ№дәҺжҹҘжүҫй—®йўҳпјҢunordered_mapдјҡжӣҙеҠ вҫјж•ҲвјҖдәӣпјҢеӣ жӯӨйҒҮеҲ°жҹҘжүҫй—®йўҳпјҢеёёдјҡиҖғиҷ‘вјҖдёӢвҪӨunordered_map

жҖ»з»“пјҡ

з®—жі•жҖқи·Ҝпјҡ

(1)nдёӘе…ғзҙ зҡ„е…ЁжҺ’еҲ—=пјҲn-1дёӘе…ғзҙ зҡ„е…ЁжҺ’еҲ—пјү+пјҲеҸҰдёҖдёӘе…ғзҙ дҪңдёәеүҚзјҖпјүпјӣ

(2)еҮәеҸЈпјҡеҰӮжһңеҸӘжңүдёҖдёӘе…ғзҙ зҡ„е…ЁжҺ’еҲ—пјҢеҲҷиҜҙжҳҺе·Із»ҸжҺ’е®ҢпјҢеҲҷиҫ“еҮәж•°з»„пјӣ

(3)дёҚж–ӯе°ҶжҜҸдёӘе…ғзҙ ж”ҫдҪңз¬¬дёҖдёӘе…ғзҙ пјҢз„¶еҗҺе°ҶиҝҷдёӘе…ғзҙ дҪңдёәеүҚзјҖпјҢе№¶е°Ҷе…¶дҪҷе…ғзҙ з»§з»ӯе…ЁжҺ’еҲ—пјҢзӯүеҲ°еҮәеҸЈпјҢеҮәеҸЈеҮәеҺ»еҗҺиҝҳйңҖиҰҒиҝҳеҺҹж•°з»„пјӣ

з»„еҗҲпјҲclassпјҢйҖ’еҪ’пјү

еҸҜйҮҚеӨҚз»„еҗҲпјҲиҝҗиЎҢPassпјү

  • е…Ғи®ёе…ғзҙ йҮҚеӨҚзҡ„дёҖзұ»з»„еҗҲй—®йўҳ

  • з”ЁP(n, k)иЎЁзӨәд»ҺnдёӘе…ғзҙ дёӯйҖүеҮәkдёӘе…ғзҙ пјҲе…Ғи®ёйҮҚеӨҚпјүзҡ„з»„еҗҲй—®йўҳпјҢйӮЈд№ҲжӯӨй—®йўҳеҸҜд»ҘеҲҶи§ЈдёәдёӨдёӘеӯҗй—®йўҳпјҡP(n, k-1)е’ҢP(n-1, k)пјҢ

  • и§ЈйҮҠеҰӮдёӢпјҡ

    • P(n, k)дёӯnдёӘе…ғзҙ зҡ„жүҖжңүkе…ғзҙ з»„еҗҲеҸҜд»ҘеҲҶжҲҗдёӨйғЁеҲҶгҖӮ

    • з¬¬дёҖйғЁеҲҶзҡ„жҜҸдёӘз»„еҗҲеқҮд»Ҙз¬¬дёҖдёӘе…ғзҙ ејҖе§ӢпјҢеҶҚиҝһжҺҘдёҠk-1дёӘе…ғзҙ зҡ„з»„еҗҲпјҢеҚіеҶҚиҝһжҺҘдёҠP(n,k-1)зҡ„жҜҸдёҖдёӘз»„еҗҲпјӣ

    • з¬¬дәҢйғЁеҲҶзҡ„жҜҸдёӘз»„еҗҲдёҚеҗ«жңүз¬¬дёҖдёӘе…ғзҙ пјҢеҚіP(n-1, k)дёӯзҡ„жҜҸдёҖдёӘз»„еҗҲпјҲжӯӨж—¶е…ғзҙ дёәеҗҺn-1дёӘе…ғзҙ пјүгҖӮ

    • еӣ жӯӨпјҢP(n, k)еҲҶи§ЈдёәP(n, k-1)е’ҢP(n-1, k)гҖӮ

еҰӮжһңз”Ёf(n, k)иЎЁзӨәP(n, k)дёӯзҡ„з»„еҗҲж•°пјҢйӮЈд№Ҳжңүпјҡ
пјҲ1пјүеҪ“k = 1ж—¶пјҢf(n, k) = n
пјҲ2пјүеҪ“n = 1ж—¶пјҢf(n, k) = 1
пјҲ3пјүеҪ“k > 1дё”n > 1ж—¶пјҢf(n, k) = f(n, k -1) + f(n-1, k)

жӯӨеӨ–пјҢжңүе…¬ејҸf(n, k) = C(n + k -1, k)пјҲиЎЁзӨәn+k-1йҖүkзҡ„з»„еҗҲж•°пјҢжІЎжңүйҮҚеӨҚе…ғзҙ пјүпјҢиҝҷеҸҜд»Ҙз”ЁеҪ’зәіжі•иҜҒжҳҺпјҢиҝҷйҮҢе°ұдёҚе•°е—ҰдәҶгҖӮ

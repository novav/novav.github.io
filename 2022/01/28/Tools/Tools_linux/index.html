lsof

`lsof`命令用于查看你进程打开的文件，打开文件的进程，进程打开的端口(`TCP、UDP`)。找回/恢复删除的文件。是十分方便的系统监视工具，因为`lsof`命令需要访问核心内存和各种文件，所以需要root`用户`执行。

lsof输出各列信息的意义如下：

  • COMMAND：进程的名称
  • PID：进程标识符
  • PPID：父进程标识符（需要指定-R参数）
  • USER：进程所有者
  • PGID：进程所属组
  • FD：文件描述符，应用程序通过文件描述符识别该文件。
  • TYPE: [DIR, REG, unknown,]

获取端口对应的进程ID=>pid
